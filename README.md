# soda_clix

## 介绍

识别和刮削媒体资源文件。

## 如何安装

### 从Docker安装

```shell
docker pull biezhihua521/soda_clix
```

### 从cargo安装

```shell
cargo install soda_clix
```

### 从Github下载二进制文件

```text
https://github.com/biezhihua/soda-resource-tools/releases/tag/v0.1.1
```

## 如何使用

### 查看当前soda_clix版本

```shell
➜  ~ soda_clix --version
soda_clix 0.1.1
```

## 开发者

### 如何编译

```shell
cargo build
```

## 注意事项

- 不支持蓝光原盘
